# SQL Server Core Internals (DBA Foundations)

This section documents hands-on DBA learning and labs focused on core SQL Server internals.

These topics form the foundation of real-world database administration and disaster recovery.

## Topics Covered
- SQL Server system databases
- Database files and file locations
- Recovery models
- Backup strategies
- Restore operations

All work was performed in a lab environment using SQL Server Management Studio (SSMS).
